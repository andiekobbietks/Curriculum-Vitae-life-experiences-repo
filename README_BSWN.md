## How to Generate PDFs (Overleaf)

1. Go to https://www.overleaf.com (free account)
2. Click "New Project" > "Upload Project"
3. Upload the `.tex` file
4. Set compiler: Menu > Compiler > **XeLaTeX**
5. Click "Recompile"
6. Download PDF
