# Wazuh Security for UnMuseum

## What is Wazuh?

Wazuh is an open-source security platform providing:

- Intrusion detection
- Log analysis
- Vulnerability assessment
- Compliance monitoring
- File integrity monitoring

**Source:** https://wazuh.com
