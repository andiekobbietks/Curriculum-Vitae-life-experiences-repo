## Supported File Types

- **Images**: Screenshots, photos, diagrams (.jpg, .png, .gif)
- **Documents**: PDFs, office documents (.pdf, .docx, .xlsx)
- **Audio**: Voice notes (if relevant) (.opus, .mp3)
- **Other**: Any other media shared in chats

## Naming Convention

```
YYYY-MM-DD_[source-chat]_[description].[ext]

Examples:
2026-01-15_project-phoenix_architecture-diagram.png
2026-01-20_client-feedback_requirements-doc.pdf
2025-12-10_team-chat_screenshot-success-metrics.jpg
```
